MARKETING FUNNEL & CONVERSION PERFORMANCE ANALYSIS

Project Overview

This project analyzes a real-world marketing funnel to identify conversion drop-offs, evaluate channel performance, and uncover opportunities to improve lead-to-customer conversion rates.
Using the Olist Marketing Funnel dataset, this analysis simulates a real business scenario where data-driven decisions are used to optimize marketing and sales strategies.

Objectives

- Analyze the marketing funnel from Lead → Customer
- Identify conversion drop-offs
- Evaluate channel performance
- Provide actionable recommendations to improve conversions

Dataset
The dataset used in this project is sourced from Kaggle:
- Olist Marketing Qualified Leads Dataset
- Olist Closed Deals Dataset

These datasets include:

- Lead information (marketing-qualified leads)
- Closed deal records (converted customers)
- Acquisition channels (origin)
- Landing page data

Tools & Technologies

- Python (Pandas, NumPy)
- Power BI / Tableau (for visualization)

Data Preparation

- Merged datasets using "mql_id"
- Created a conversion flag to identify customers
- Cleaned missing values and standardized fields

Key Metrics

- Total Leads
- Total Customers
- Conversion Rate
- Drop-off Rate
- Channel-wise Conversion Performance

Analysis & Findings

🔹 Funnel Performance

- Significant drop-off observed between Leads and Customers
- Indicates potential issues in lead quality or sales conversion process

🔹 Channel Performance

- Some channels generated high lead volume but low conversions
- Other channels produced fewer but higher-quality leads

🔹 Bottleneck Identification

- The biggest bottleneck exists at the conversion stage (Lead → Customer)

Visualization
This project includes:
- Funnel chart (Lead -> Customer)
- Channel-wise conversion bar charts
- Summary performance tables

This project demonstrates how data analytics can be used to identify inefficiencies in the marketing funnel and provide actionable strategies to improve overall conversion performance.
